# Stock Trading Platform

This is a full-stack stock trading platform inspired by Zerodha, built to simulate basic trading workflows such as portfolio tracking, watchlists, and data visualization.

The project is structured into separate frontend, dashboard, and backend applications. The frontend handles the landing interface, the dashboard provides the main user interaction layer, and the backend manages APIs and database operations.

Key features include:
- Positions and holdings tracking  
- Watchlist functionality  
- Interactive dashboard with charts  
- Demo authentication system (username and password based)  
- REST API communication between client and server  

The application is built using React for the frontend and dashboard, Node.js with Express for the backend, and MongoDB for data storage. Axios is used for API communication, and Chart.js is used for data visualization.

This project was built to understand how a full-stack application works in practice, including how the frontend communicates with the backend, how data is managed, and how dashboards are structured.

Project structure:
- `/frontend` – landing page  
- `/dashboard` – main application UI  
- `/backend` – server and database logic  <br>

Project Setup:

Clone the repository: https://github.com/tahirainam/Stock-Trading-Platform.git

Front-end:
`
cd frontend
npm install
npm start
`

Dashboard:
`
cd dashboard
npm install
npm start
`

Backend:
`
cd bakend
npm install
npm start
`
